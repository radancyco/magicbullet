Magic Bullet is a script importer. Its sole purpose is to load other scripts when needed. On its own, the script will do nothing until you start passing parameters into it. This allows a developer to load different scripts in for different client needs. Managing scripts in this manner allows development to maintain website enhancements from a single location. The more websites we include Magic Bullet on, the more leverage the development team will have in addressing unforeseen future needs (or issues) that may arise across multiple websites.

Properties

Use the following properties to configure this component.

Anchor (Attributes) Attributes
Attribute Note Description
id String, Mandatory An id attribute with a value of tmp-magic-bullet is required. This value is needed for vital functions within the script.
src String, Mandatory A src attribute with a value of https://services.tmpwebeng.com/magicbullet/ is required.
data-* String, Optional Data attributes are what load the desired scripts. All data attributes, when used, must always equal true.

Scripts

The following scripts are available to use in Magic Bullet. You may visit each page listed below to learn more about them.
