Properties

Use the following properties to configure this component.

Anchor (Attributes) Attributes
Attribute Note Value Description
id String, Mandatory tmp-magic-bullet This value is required for critical functions within the script.
src URL, Mandatory https://services.tmpwebeng.com/magicbullet/ Please note that on the runmytests.com and talentbrew.com domains, the QA version of the script will run, which lives on this website.
data-a11y String, Optional true Learn more about Accessibility Patch
data-alert String, Optional true Learn more about System Alert
data-ccpa String, Optional true Learn more about California Consumer Privacy Act (CCPA) Notice
data-gdpr String, Optional true Learn more about General Data Protection Regulation (GDPR) Notice
