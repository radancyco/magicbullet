Introduction

Magic Bullet is a script importer. Its sole purpose is to load other scripts when needed. On its own, the script will do nothing until you start passing parameters into it. This allows a developer to load different scripts in for different client needs. For example, let's say we want to address accessibility issues on a client's website? Well, there is a script for that. To load it, all we would need to do is add the following parameter to the Magic Bullet script element:

<script id="tmp-magic-bullet" data-a11y="true" src="https://services.tmpwebeng.com/magicbullet/"></script>

This will append and initiate the following:

<script id="a11y-fixes" src="https://services.tmpwebeng.com/magicbullet/a11y/prod/"></script>

Note: On runmytests.com and talentbrew.com, the QA version of the script will load: https://services.tmpwebeng.com/magicbullet/a11y/qa/

You can even load multiple scripts in:

<script id="tmp-magic-bullet" data-a11y="true" data-gdpr="true" src="https://services.tmpwebeng.com/magicbullet/"></script>

Why?

Managing scripts in this manner allows development to maintain website enhancements from a single location. The more websites we include Magic Bullet on, the more leverage the development team will have in addressing unforeseen future needs (or issues) that may arise across multiple websites.

Parameters

The following is a list of parameters that can be passed to the script.

id

An id attribute with a value of "tmp-magic-bullet" is required. This value is needed for vital functions within the script. This parameter is mandatory.

<script id="tmp-magic-bullet" src="https://services.tmpwebeng.com/magicbullet/"></script>

Return to top

src

A src attribute with a value of "https://services.tmpwebeng.com/magicbullet/" is required. This parameter is mandatory.

<script id="tmp-magic-bullet" src="https://services.tmpwebeng.com/magicbullet/"></script>

Return to top

data-*

Data attributes are what load the desired scripts. All data attributes, when used, must always equal "true". This parameter is optional.

<script id="tmp-magic-bullet" data-a11y="true" src="https://services.tmpwebeng.com/magicbullet/"></script>
