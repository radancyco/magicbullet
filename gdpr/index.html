This script may be used to generate a General Data Protection Regulation (GDPR) consent notice on any TMP hosted website. Due to privacy concerns, European Union (EU) citizens who engage with any website that collects data, via cookies, must now be warned about this practice in advance and be able to (implicitly or explicitly) opt-in to permit this data to be used. You can learn more about the GDPR on the official website.

Note: This document is for internal TMP Worldwide use only.

Implementation

To adhere to this regulation, a notice must be included on every page of your site. The following script may be implemented on a site to achieve this goal. Which way you choose to implement the script, largely depends on the size of your site and/or how it has been templated. In most cases, implementation should not take more than 30-60 minutes per site.

HTML

If you require HTML, simply add the following script before the closing body tag of your page or template:

<script id="tmp-magic-bullet" data-gdpr="true" data-gdpr-client-name="[client name]" data-gdpr-policy-url="[site policy url]" src="https://services.tmpwebeng.com/magicbullet/"></script>

JavaScript

If your site contains many templates, but leverages the same, single JavaScript file, implementing a sitewide notice may be more easily achieved by adding the following DOM method to your global script: 

var magicBullet = document.createElement("script");
magicBullet.setAttribute("id", "tmp-magic-bullet");
magicBullet.setAttribute("data-gdpr", "true");
magicBullet.setAttribute("data-gdpr-client-name", "[client name]");
magicBullet.setAttribute("data-gdpr-policy-url", "[site policy url]");
magicBullet.setAttribute("src", "https://services.tmpwebeng.com/magicbullet/");
document.body.appendChild(magicBullet);

Note: The data-gdpr, data-gdpr-client-name and data-gdpr-policy-url attributes are mandatory parameters.

Parameters

The following is a list of parameters that can be passed to the script to exhibit more control over the notice.

Mandatory

The following parameter(s) are mandatory.

Script Initiation

Passing this data attribute a value of "true", will initiate the GDPR script.

HTML

data-gdpr="true"

JavaScript

magicBullet.setAttribute("data-gdpr", "true");

Client Name

Passing this data attribute a string value, such as the name of the client, will include that value in any needed legal verbiage, such as the notice placed within TalentBrew's Data Collection Forms.

HTML

data-gdpr-client-name="Clarity Robotics"

JavaScript

magicBullet.setAttribute("data-gdpr-client-name", "Clarity Robotis");

Policy URL

Passing this data attribute a policy URL, will include that URL within the notice and inline form message. The value must be in the form of a fully qualified URL. Protocol-relative URL's should not be used. If the privacy policy is local to the domain, then please use a relative path instead—e.g., "/privacy/" instead of "https://www.clarityrobotics.com/privacy/".

HTML

data-gdpr-policy-url="https://www.clarityrobotics.com/privacy/"

JavaScript

magicBullet.setAttribute("data-gdpr-policy-url", "https://www.clarityrobotics.com/privacy/");

Formatting

Warning: Please do not attempt to override the banner notice with CSS or JavaScript at the implementation level. Local overrides may interfere with future formatting changes required within the GDPR script itself. If you run into any formatting issues that can't be solved with any available parameters the GDPR script offers, then please contact Michael Spellacy for assistance. The inline form message may be safetly manipulated with CSS.

The following parameter(s) can be used to apply basic formatting to the notice:

Button Color

Passing this data attribute any valid CSS color value will change the background color of the "Accept" button. This parameter is optional.

HTML

data-gdpr-button-color="lavender"

JavaScript

magicBullet.setAttribute("data-gdpr-button-color", "lavender");

Button Text

Passing this data attribute any string value will change the text of the "Accept" button. The default text is "Accept". This parameter is optional.

HTML

data-gdpr-button-text="OK"

JavaScript

magicBullet.setAttribute("data-gdpr-button-text", "OK");

Button Text Color

Passing this data attribute any valid CSS color value will change the text color of the "Accept" button. This parameter is optional.

HTML

data-gdpr-button-color-text="lime"

JavaScript

magicBullet.setAttribute("data-gdpr-button-color-text", "lime");

Font Size

Passing this data attribute any valid CSS unit, will change the font size used within the notice. This parameter is optional.

HTML

data-gdpr-font-size="18px"

JavaScript

magicBullet.setAttribute("data-gdpr-font-size", "18px");

Notice Background Color

Passing this data attribute any valid CSS color value will change the background color of the notice. This parameter is optional.

HTML

data-gdpr-notice-color="tomato"

JavaScript

magicBullet.setAttribute("data-gdpr-notice-color", "18px");

Notice Text Color

Passing this data attribute any valid CSS color value will change the text color of the notice. This parameter is optional.

HTML

data-gdpr-notice-text-color="purple"

JavaScript

magicBullet.setAttribute("data-gdpr-notice-text-color", "purple");

Text Alignment

Passing this data attribute any valid CSS text-align value, will change the alignment of the text within the notice. The default value is "center". This parameter is optional.

HTML

data-gdpr-text-align="right"

JavaScript

magicBullet.setAttribute("data-gdpr-text-align", "right");

Z-Index

Passing this data attribute any number from 1 to 2147483647, will change the z-index stack. The default value is "10000". This parameter is optional.

HTML

data-gdpr-z-index="12000"

JavaScript

magicBullet.setAttribute("data-gdpr-z-index", "12000");

Behavior

The following parameter(s) can be used to apply or remove basic behaviors to the notice:

New Window

Passing this data attribute a value of "false" will disable the privacy policy link from opening in a new window. The default behavior is for all notice links to open in a new window. The visual cue associated with these links is a usability best practice. These cues will not appear when new windows are disabled. This parameter is optional.

HTML

data-gdpr-new-window="false"

JavaScript

magicBullet.setAttribute("data-gdpr-new-window", "false");

TalentBrew Data Collection Forms

The second task of this script is to include a privacy policy statement on all TalentBrew forms that collect data. By default, the script will look for the .data-form form-field.submit class and prepend the following element, and message, before the form's "Submit" button:


<p class="form-field gdpr-eu-tmp-notice">By submitting your information, you acknowledge that you have read our privacy policy (this content opens in new window) and consent to receive email communication from [company name].</p>

If you need to override the default behavior, then you may include the following data attribute and value.

HTML

data-gdpr-form-bypass="true"

JavaScript

magicBullet.setAttribute("data-gdpr-form-bypass", "true");

Removing the default behavior means that you may need to manually insert the message into the desired position withn the form. To do this, simply place the following markup wherever you would like the message to appear. The surrounding p tag can be replaced with a span or any other tag of your choosing, but please be sure to keep the gdpr-eu-tmp-notice class name in place, as the message will still be inserted into the tag, via JavaScript. The "form-feild" class may be removed, if needed.

Note: CSS formatting may be safetly applied to this tag, as needed.

<p class="form-field gdpr-eu-tmp-notice"></p>
